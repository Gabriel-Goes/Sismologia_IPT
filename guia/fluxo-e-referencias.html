Fluxo E Referencias¶

Esta pagina segue a logica:

  • User Guide -> API (objetos Python)

  • API -> [source] (codigo-fonte em _modules)

  • Source -> [docs] (retorno para a pagina de API)

Aquisicao e normalizacao de eventos¶

O fluxo de entrada usa:

  • pre-processamento (source): analise_dados/pre_processa.py;

  • entrypoint: analise_dados.pre_processa.main.

Depois, a aquisicao usa:

  • script (source): nucleo/fluxo_eventos.py;

  • funcao principal: nucleo.fluxo_eventos.fluxo_eventos;

  • iteracao de picks: nucleo.fluxo_eventos.iterar_eventos;

  • utilitarios: nucleo.utils.csv2list e nucleo.utils.DualOutput.

Classificacao (RNC)¶

A classificacao na RNC usa:

  • script (source): rnc/run.py;

  • entrada do fluxo: rnc.run.main;

  • extracao de espectros: rnc.data_process.spectro_extract;

  • FFT: rnc.data_process.get_fft;

  • inferencia: rnc.prediction.discrim.

Pos-processamento e mapas¶

As etapas finais usam:

  • pos-processamento (source): analise_dados/pos_processa.py;

  • limpeza/consolidacao: analise_dados.pos_processa.clean_data;

  • consolidacao por evento: analise_dados.pos_processa.recall_event;

  • mapas (source): analise_dados/gera_mapas.py;

  • entrypoint de mapas: analise_dados.gera_mapas.main;

  • mapa de predicoes: analise_dados.gera_mapas.plot_pred_map;

  • macro-regioes: analise_dados.gera_mapas.plot_macroregions.

Relatorio¶

Para fechamento do pipeline:

  • script (source): mapa.py;

  • docs: mapa.generate_map_latex.
