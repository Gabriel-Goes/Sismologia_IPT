Operacao 0024 - Organizacao da fase de testes de execucao
==========================================================

Resumo
------

Formaliza a transicao da fase de leitura/compreensao do legado para a fase de
teste de execucao do codigo.

Efeito principal
----------------

- introduz marcacao sintetica por operacao com ``Fase`` e ``Tag``;
- padroniza classificacao para proximas operacoes de teste;
- define sequencia de execucao local sem aquisicao (sem ``-e``) antes dos
  testes de conectividade no IAG.
