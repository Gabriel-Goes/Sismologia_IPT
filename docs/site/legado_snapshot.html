Snapshot Legado¶

Uma copia congelada da documentacao legado foi arquivada nesta branch para consulta historica durante a refatoracao cleanroom.

Publicacao no GitHub Pages¶

O snapshot pode ser publicado como subsite no GitHub Pages, abaixo da documentacao principal (Alpha), por exemplo em /legado/. Isso permite que o leitor navegue pelo material antigo sem precisar trocar de branch.

Localizacao do snapshot no repositorio:

  • docs/legacy_snapshot/README.md

  • docs/legacy_snapshot/sphinx_source/

  • docs/legacy_snapshot/operacoes_md/

  • docs/legacy_snapshot/documentacao_raiz/

Escopo:

  1. preservar rastreabilidade das decisoes e operacoes do ciclo anterior;

  2. evitar dependencia de navegacao em branches antigas para leitura tecnica.
