Operacao 0026 - Protocolo de importacao test_matrix
===================================================

Resumo
------

Isola a evolucao de protocolo de documentacao em operacao propria, separada das
operacoes de teste de execucao.

Efeito principal
----------------

- cria o script ``scripts/dev/import_test_matrix_to_docs.sh``;
- automatiza importacao de anexos de ``test_matrix`` para Sphinx;
- remove necessidade de montagem manual dos anexos por operacao.
