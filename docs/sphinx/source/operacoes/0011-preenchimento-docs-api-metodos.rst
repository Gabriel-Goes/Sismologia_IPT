Operacao 0011 - Preenchimento dos docs de metodos da API
=========================================================

Resumo
------

As paginas de API foram preenchidas com explicacao de comportamento de cada
metodo/classe, indo alem da assinatura automatica.

Efeito principal
----------------

- melhora leitura tecnica por humanos (orientacao e revisao);
- mantem navegacao bidirecional ``[docs] <-> [source]``;
- evita perda de conteudo manual por sobrescrita automatica do autosummary.

Arquivos de interesse
---------------------

- ``docs/sphinx/source/conf.py``
- ``docs/sphinx/source/api/generated/*.rst``
