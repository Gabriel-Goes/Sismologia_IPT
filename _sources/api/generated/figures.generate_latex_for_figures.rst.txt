figures.generate_latex_for_figures
==================================

.. currentmodule:: figures

.. autofunction:: generate_latex_for_figures

Resumo
------

Varre o diretorio de figuras do pos-processamento e monta automaticamente um
bloco LaTeX com figuras para inclusao no relatorio preditivo.

Parametros
----------

Esta funcao nao recebe parametros.

Retorno
-------

Nao retorna valor.

Efeitos colaterais
------------------

- Escreve ``figures.tex`` no template do relatorio em
  ``fonte/relatorio-sismologia/tex/relatorio_preditivo/tex/``.
