Operacao 0014 - Padrao Verde para API e Source do fluxo legado
===============================================================

Resumo
------

Migracao da documentacao do legado para o padrao Verde:
``User Guide -> API docs <-> source (_modules)``.

Efeito principal
----------------

- scripts Python deixam de ser tratados como artefatos;
- entrypoints principais passam a ter pagina de API e ``[source]``;
- links de narrativa apontam para docs e source internos do Sphinx.

Arquivos de interesse
---------------------

- ``docs/sphinx/source/visao_geral.rst``
- ``docs/sphinx/source/guia/fluxo-e-referencias.rst``
- ``docs/sphinx/source/api/index.rst``
- ``docs/sphinx/source/api/generated/*.rst``
- ``docs/sphinx/source/conf.py``
- ``docs/sphinx/source/artefatos/index.rst``
- ``skills/docs-verde-style/``
