Operacao 0009 - Estrutura Sphinx API/source e artefatos renderizados
=====================================================================

Resumo
------

Consolidacao de uma arquitetura de documentacao navegavel no padrao
User Guide -> API -> [source] <-> [docs], com renderizacao de scripts Bash e
arquivos TXT diretamente no GitHub Pages.

Efeito principal
----------------

- cria ponte entre explicacao de fluxo e referencia tecnica de funcoes;
- habilita rastreabilidade bidirecional entre documentacao e codigo;
- substitui links de download por paginas renderizadas para artefatos textuais.

Arquivos/diretorios de interesse
--------------------------------

- ``docs/sphinx/source/conf.py``
- ``docs/sphinx/source/guia/``
- ``docs/sphinx/source/api/``
- ``docs/sphinx/source/artefatos/``
- ``docs/sphinx/source/operacoes/0008-preparacao-ambiente-pyenv-duplo.rst``
- ``.github/workflows/docs.yml``
