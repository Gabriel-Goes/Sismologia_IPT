# Seismic Event Discriminator

Refatoracao cleanroom do classificador sismologico para discriminacao entre eventos naturais e antropicos.

## Objetivo

Reescrever o fluxo do zero com foco em:

- arquitetura clara e modular;
- reproducibilidade cientifica;
- documentacao tecnica para orientacao academica;
- testes e validacao incremental por etapa.

## Estrutura inicial

- `src/seismic_event_discriminator/`: codigo fonte do novo fluxo.
- `scripts/`: wrappers de execucao e automacao.
- `docs/sphinx/`: documentacao Sphinx do projeto v3.

## Documentacao

A documentacao Sphinx desta branch fica em `docs/sphinx/source/`.
